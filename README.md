## ✨ Características

| Característica | Descripción |
|----------------|-------------|
| 💾 **Guardado Automático** | Cada mensaje se guarda solo - ¡No hay botón de guardar! |
| 📚 **Múltiples Conversaciones** | Crea nuevas sin perder las anteriores |
| 🔗 **Persistente** | Recarga la página o cierra el navegador - tus chats siguen ahí |
| 🎭 **Personalidad Única** | El asistente elige su propio nombre |
| 🤖 **7 Modelos de IA** | Elige entre diferentes modelos según tu necesidad |
| 🗑️ **Gestión Simple** | Borra conversaciones individuales o todas |
| 🔒 **Privada** | Cada usuario tiene su propio espacio |
| ⚡ **Rápido** | Respuestas en tiempo real |
